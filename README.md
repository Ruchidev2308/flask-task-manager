# 🐍 Flask Task Manager Backend

A web application built using Python Flask, HTML, and CSS. 

### 🌟 Features Applied
- **Data Structures:** Stored task information dynamically using a Python list of dictionaries.
- **File I/O:** Saves tasks permanently into a local `tasks.json` file.
- **Conditional Logic:** Evaluates text submission data and toggles completion status properties.
