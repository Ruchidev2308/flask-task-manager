 🐍 Flask Task Manager Backend

A clean, responsive, dark-themed web application built using Python Flask, HTML, and CSS. This project combines core Python storage concepts with dynamic frontend components.

## Core Concepts Applied

- **Variables & Data Types:** Controlled app state using strings, integers, and boolean flags (`True`/`False`).
- **Conditionals (Lecture 2):** Evaluates form data and toggles completion status properties dynamically.
- **Lists & Dictionaries (Lecture 3 & 4):** Structured task lists using clear, nested key-value dictionary elements.
- **Loops (Lecture 5):** Scans the internal dataset memory lists to match specific ID entries for toggle or drop operations.
- **File I/O (Lecture 7):** Saves tasks permanently into a local `tasks.json` file so data persists across server restarts.

##  Tech Stack Used

- **Backend:** Python 3, Flask framework, JSON storage
- **Frontend:** HTML5, CSS3

##  How to Run Locally

1. Clone this repository onto your machine.
2. Install the requirements:
   ```bash
   pip install flask
   ```
3. Run the application:
   ```bash
   python app.py
   ```
4. Open your web browser and navigate to: `http://127.0.0.1:5000`
